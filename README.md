## 🛠️ Technologies Used
This project is built using standard front-end web technologies:
* **HTML5:** Used for structuring the web content (`index.html`).
* **CSS3:** Used for styling, theme design, and aesthetics (`fashion.css`).
* **JavaScript:** Used for adding interactivity and dynamic features to the site.
